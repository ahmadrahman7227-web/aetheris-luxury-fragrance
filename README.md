# AETHERIS — Luxury Cinematic Fragrance Website

AETHERIS is a cinematic luxury fragrance landing page built with React, Vite, Tailwind CSS, GSAP, and editorial-inspired design systems.

This project was designed to create an immersive luxury storytelling experience inspired by:
- High-end fragrance campaigns
- Fashion editorials
- Cinematic visual composition
- Luxury brand aesthetics
- Awwwards-level interactive experiences

---

AETHERIS is not just a product landing page.
